### Визуализация фрактала комплексного обобщения функции Коллатца

#### Математическая основа

Исходная дискретная функция Коллатца для целых чисел $n \in \mathbb{Z}$:

$$
f(n) = 
\begin{cases}
n/2 & \text{если } n \equiv 0 \pmod{2} \\
3n + 1 & \text{если } n \equiv 1 \pmod{2}
\end{cases}
$$

**Непрерывное комплексное обобщение** для $z \in \mathbb{C}$:

$$
F(z) = \frac{7z + 2 - \cos(\pi z)(5z + 2)}{4}
$$

#### Алгоритм визуализации

##### Итерационный процесс

Для каждой точки комплексной плоскости $z_0 = x + iy$ вычисляем последовательность:

$$
z_{n+1} = F(z_n) = \frac{7z_n + 2 - \cos(\pi z_n)(5z_n + 2)}{4}
$$

##### Критерий

**Радиус**: $R = 10^6$

**Условие остановки**:
- Если $|z_n| > R$ - точка считается "убежавшей"
- Если достигнуто максимальное число итераций $N_{\text{max}}$ - точка считается остающейся

**Мы красим точки в зависимости от того, сколько шагов (итераций) вычислений они выдерживают, прежде чем "сбегут" (их модуль станет больше escape-радиуса).**

Цвета распределяются по спектру от теплых к холодным:
- **Золотой / Оранжевый** — точки, которые сбегают быстро (мало итераций).
- **Розовый / Фиолетовый** — точки, которые сбегают медленно (много итераций).
- **Тёмно-синий** — точки, которые не сбегают вообще (максимальное число итераций).

**Условие остановки:** вычисления прекращаются, как только модуль числа превышает escape-радиус **ИЛИ** достигнуто максимальное количество итераций.
